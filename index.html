Widget

  1. AspectRationLayout

    Demo

    Set views to always maintain a set aspect ratio

  2. ButtonWithFont

    GitHub

    Easily set a custom font to the text of a button via XML

  3. CancelEditText

    Website · Demo

    Easily set a custom drawable to clear out an EditText field

  4. CheckBoxWithFont

    Website · Demo

    Easily set a custom font to a CheckBox

  5. CustomCircularProgressBar

    Demo

    Allows you to have custom circular loading animations

  6. CustomProgressDialog

    Demo

    Allows you to set a custom loading animation in a dialog.

  7. GradientTextView

    Demo

    Allows you to easily display a TextView with a gradiant

  8. ResizedTextView

    Website · Demo

    TextView that will easily scale based on your size constraints, and the amount of text used.

  9. SpreadsheetView

    Demo

    Easily create a large "spreadsheet" for displaying data. This is popular for sports scores, for example.

  10. TextViewWithFont

    Website · Demo

    Easily set a custom font to a TextView in XML.

ListView

  1. SectionListView

    Website · Demo

    Allows you to easily create a listview with sections, similar style to that of the Contacts app.

ViewPagers

  1. VerticalViewPager

    Website · GitHub

    A ViewPager that scrolls vertically, instead of the typical horiztonal scroll.

  2. ParentSwipingViewPager

    Website · GitHub

    A ViewPager that helps with touch detection. This is helpful when using a NavDrawer and ViewPager for example.

SVG

  1. WTALogo

  2. FishBowlLogo

    Website · GitHub

Images

    GitHub

    Website · GitHub

    Allows you to quickly change all fonts in a view.

  1. FontTypeFaceSpace

  2. IsPackageInstalled

    Website · Demo

    Simple, flexible and lightweight configuration switching.
